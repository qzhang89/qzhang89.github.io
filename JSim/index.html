Welcome!

In this website, we design several computational simulations using JavaScript. Applications are based on fundamental Quantum Mechanics.

We visualize data on our web pages using the <canvas> tag, which is a new element specified in HTML5. Therefore, to view our web pages, your browser should support the <canvas> tag.
